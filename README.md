# 💰 Sistema de Gestão Financeira

Este projeto consiste em um sistema de gestão financeira simples, desenvolvido com foco em organização de despesas, receitas e controle financeiro básico.

## 🧱 Tecnologias Utilizadas

### Backend

* Java
* Spring Boot
* API REST

### Frontend

* HTML
* CSS
* JavaScript

---

## 📁 Estrutura do Projeto

```
/meu-projeto
  /backend
    /src
      /main
        /java
        /resources
  /frontend
    index.html
    style.css
    script.js
```

---

## ⚙️ Funcionalidades

* Cadastro de receitas
* Cadastro de despesas
* Listagem de transações
* Cálculo de saldo
* Organização básica de dados financeiros

---

## 🚀 Como Executar o Projeto

### Backend (Spring Boot)

1. Acesse a pasta do backend:

   ```
   cd backend
   ```

2. Execute o projeto:

   * Pela IDE (IntelliJ)
   * Ou via terminal:

     ```
     ./mvnw spring-boot:run
     ```

3. O servidor iniciará em:

   ```
   http://localhost:8080
   ```

---

### Frontend

1. Acesse a pasta do frontend:

   ```
   cd frontend
   ```

2. Abra o arquivo `index.html` no navegador
   ou utilize uma extensão como Live Server no VS Code.

---

## 🔗 Integração

O frontend consome a API do backend através de requisições HTTP (fetch/AJAX), utilizando endpoints expostos pelo Spring Boot.
